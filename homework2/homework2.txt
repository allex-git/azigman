#Поясніть роль і значення кожного з наступних компонентів в архітектурі ОС Linux:
#
#a) Kernel
#Kernel – ядро ОС, яке керує апаратним забезпеченням, розподіляє ресурси (процесор, пам’ять, пристрої введення/виведення), забезпечує роботу драйверів та системні виклики для взаємодії програм з обладнанням.
#
#b) Libraries
#Libraries – набори готових функцій, які програми викликають для роботи з файлами, мережею, графікою тощо, отримуючи стандартизований доступ до можливостей системи без необхідності писати власну реалізацію.
#
#c) System utilities
#System utilities – користувацькі та адміністративні інструменти, що взаємодіють з ядром через системні виклики для керування файлами, процесами, налаштуваннями та станом системи.
